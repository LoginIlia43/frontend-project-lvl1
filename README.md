### About:
The package contains 5 mathematical mini-games.

### Installation:
Download or clone the package from the repository;

To install: 
```
$ npm link 
```

### How to start:
Write one of the following commands:
```
$ Brain-even
$ Brain-calc
$ Brain-gcd
$ Brain-progression
$ Brain-prime
```
